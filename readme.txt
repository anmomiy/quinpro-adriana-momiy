Proyecto creado por Adriana Momiy para el curso de Desarrollo Web. 

**Detalles del Sitio** 
Nombre de la empresa: Quinpro SAC 
Descripción breve: Es una microempresa peruana que se dedica a la venta de carne congelada.  
Objetivo del sitio: El objetivo del sitio solo es mostrar los productos y brindar la información de contacto. 
Posteriormente, al llevar el curso de JavaScript y React se convertirá en un e-commerce. 

Otros datos: La creación de este sitio tiene solo fines educativos. 
Se decidió usar la empresa como referencia para el sitio ya que trabajo ahí pero, por el momento, no hay planes de que el negocio lo utilice.

**Herramientas Utilizadas**
HTML
CSS
Bootstrap: Solo para cosas puntuales como la barra de navegación y el carrusel. 
SASS: Para ordenar el código y usar variables. 
Librerías de animación: AOS (animate on scroll).
